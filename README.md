# 🎉 Event Registration System

A modern **Full Stack Django Web Application** for managing event registrations and participant records.

This project was developed using **Django**, **Python**, **SQLite (Development)** and is designed to be easily migrated to **MySQL on Amazon RDS** and deployed on **AWS EC2**.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/SanjayGhosh93/Event-Registration-System.git
```

---

## Go to Project Folder

```bash
cd Event-Registration-System
```

---

## Create Virtual Environment

Windows

```bash
python -m venv venv
```

Activate

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install django
```

or

```bash
pip install -r requirements.txt
```

---

## Apply Migrations

```bash
python manage.py makemigrations

python manage.py migrate
```

---

## Run Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
